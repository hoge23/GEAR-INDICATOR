# GearShiftIndicator

ホールセンサーでシフトノブの磁石位置を検出し、現在のギアポジションをLEDドットマトリクスに表示するArduinoプロジェクトです。

OBD2方式と違い、停車中・半クラ時でも正確なギアポジションを表示できます。

## 配線

| Arduino | 接続先 |
|---------|--------|
| D2 | LEDマトリクス DIN |
| D3 | LEDマトリクス CS |
| D4 | LEDマトリクス CLK |
| D6 | イルミ信号（フォトカプラ経由） |
| A0〜A3 | ホールセンサー 1〜4 |
| 5V / GND | 各モジュールの電源 |

電源はUSBから供給します。同じUSBポートにPCを接続すればシリアルモニタでデバッグ可能です。

## 仕組み

```
シフトノブ（磁石）
      ↓
ホールセンサー × 4
      ↓
Arduino（ギアポジション推定）
      ↓
LEDドットマトリクス（表示）
```

シフトノブに固定したネオジム磁石の位置を4つのホールセンサーで読み取り、ニュートラル時との差分ベクトルを基準に最近傍のギアポジションを推定します。

## 使い方

### 1. 書き込み

Arduino IDEで `gear_shift_indicator.ino` を開き、Arduinoに書き込みます。

### 2. 校正

シフトノブをニュートラルに入れた状態でシリアルモニタ（9600bps）を開き、`RAW:` の値を確認します。

```
RAW: 409,420,395,427 | DIFF: 0,0,0,0 | GEAR: N
```

この値をコード内の `NEUTRAL_BASELINE` に設定してください。

```cpp
const int NEUTRAL_BASELINE[NUM_SENSORS] = {409, 420, 395, 427};
```

### 3. ギアテーブルの調整

各ギアに入れた状態でシリアルモニタの `DIFF:` の値を確認し、`GEAR_TABLE` に設定します。

```cpp
const GearPoint GEAR_TABLE[] PROGMEM = {
  {GEAR_1, {-143,  73,  -2,  58}},
  {GEAR_2, {  90,-173,  84, -28}},
  // ...
};
```

ギアポジションが安定しない場合は `DIST_THRESHOLD` を調整してください。

```cpp
const long DIST_THRESHOLD = 30000; // この値を大きくすると判定が緩くなる
```

### 4. 輝度の自動切り替え

イルミ信号をフォトカプラ経由でD6に入力すると、昼夜で自動的に輝度が切り替わります。不要な場合はコードから該当行を削除してください。

```
RAW: 409,420,395,427 | DIFF: 0,0,0,0 | GEAR: N
```

## 既知の問題・改善点

- ドットマトリクスの視認性がやや低い。次作るなら14セグLEDを使いたい
- インジケーターの固定場所によっては夜間にフロントガラスへ映り込む場合がある
- ホールセンサーの温度・経年ドリフトが発生した場合は再校正が必要
